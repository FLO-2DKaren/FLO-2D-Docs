Computer set-up

Overview

The Admin IT should set up the computer with these:

  • FLO-2D Setup.exe (subscribers)

  • FLO-2D Activator.exe (subscribers)

  • QGIS *.MSI installer (all users)

The end user can install these:

  • Get the FLO-2D Gila Plugin (all users)

If you are not a subscriber, take the class using the FLO-2D_Demo software that is packaged inside the class installer.

This class was designed for FLO-2D Gila Plugin 1.1.0.

Required Data

The required data is on the FLO-2D Flash Drive or if was installed with this document:

Step 1: Set up QGIS

  1. Open QGIS

  2. Version 3.34 to 3.42 should work with this tutorial. Use the search bar or the desktop folder.

  1. Open the Setting/Options menu.

  1. Find the CRS tab and check the Use Project CRS button.

Step 2: Add plugins

With QGIS installed it is time to add the FLO-2D Gila plugin and a few other handy plugins.

  1. Navigate to the plugin manager.

  1. Select All from the left side bar and use the filter to find and install the following:

  • QuickMapServices

  • Profile tool

  • FLO-2D Rasterizor

  • FLO-2D Map Crafter

  • Curve Number Generator

  • MapBiomas Collection

  • OSMDownloader

  • Serval

  1. Install the FLO-2D Gila Plugin. It should be in the class or download folder.

  2. Close the window once the zipped installers are finished.

  1. Click the Quick Map Services button and Click Settings.

  2. Click Get contributed pack.

  1. Change the tab to Visibility and hide all layers then check google, esri, and osm.

Step 3: Close QGIS to save profile

  1. Close QGIS to save the user profile to the computer.

  2. If you skip this step and crash QGIS you will have to repeat this lesson.
