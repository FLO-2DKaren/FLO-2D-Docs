Project Set-Up

Overview

This step will outline the process of setting up a FLO-2D project using the Plugin for QGIS. Setting up the computational domain, creating a grid, interpolating elevation data and spatially variable roughness.

Required Data

This step makes use of terrain elevation data, project domain, and roughness data. The required data is available here:

Step 1: Open QGIS and load project domain

  1. Search the start menu and run the QGIS Desktop program. Look for QGIS version 3.34 or higher.

  1. Click the New Project icon to load a new map.

  1. Open the project folder and drag the file Project Domain TD.shp onto the map space. This will set the CRS to the correct EPSG code. The code should be 29101.

Step 2: Create the GeoPackage

  1. Click the Settings icon.

  1. Fill out the dialog box as shown below. Set the Grid cell size to 30 m. Click Create.

  2. Save the geopackage file to the project folder. Name the file Tailings Dam.gpkg.

  1. Set the project CRS to SAD69 / Brazil Polyconic (m). Filter the list with an EPSG code: 29101. Select EPSG: 29101 and click OK.

  1. Wait for the geopackage to write and check the accuracy of the project settings and click OK.

  1. Click Yes when prompted to save external layers that were added to the project.

  1. Click OK in the next window

  1. The project should look like this:

  1. Click the Collapse button and then expand the Grid Tools Widget.

Step 3: Create the grid

  1. From the Grid Tools widget, click Create Grid.

  1. Select the Project Domain TD layer, and the Cell Size field and click OK. Click OK to close the Grid information window.

Step 4: Interpolate the elevation

  1. Collapse the FLO-2D Project Layers.

  2. Select the Project Domain Layer so that it is active.

  3. Drag the Elevation Pre Disaster.tif onto the map.

  1. Uncheck the Project Domain Layer and drag it to the bottom of the Layers List.

  2. Drag the Elevation layer to the bottom of the list.

  3. The map should look like the following image.

  1. To interpolate the elevation to the grid layer from a raster layer, use the Sample Grid Elevation icon.

  1. Click on the Sample Grid Elevation icon and enter the required data in the dialog fields and click OK. Click the OK box to close the Sampling Done dialog box.

  1. The Render elev button will show if the process worked correctly.

Step 5: Assign roughness

  1. To assign roughness data, open the project folder and drag the Landuse TD.shp file onto the map space.

  1. Click the Sample roughness icon.

  1. Fill the dialog box and click OK. Click OK to close the next message window.

  1. The roughness values and elevations are assigned to the grid layer in the Schematized Layers group. Right click the Grid layer and click Open Attribute Table to see the data.

  1. Click the Set Control Parameters Icon to assign the control variables.

  1. Fill the dialog box using the two figures below and save the data to the GeoPackage with the Save icon. The variable descriptions and additional instructions are presented in the Data Input Manual. The drag bar will expose the save button.

    “C:\Users\Public\Documents\FLO-2D PRO Documentation\flo_help\Manuals\Data Input Manual PRO 2023.pdf”

  1. Save the project.

Step 6: Saving the Project

  1. Click the main Save icon on the QGIS toolbar and click Yes when prompted to save external layers.

  1. Move the Elevation Pre Disaster and Landuse TD layers to the left side of the next dialog box, then click OK
